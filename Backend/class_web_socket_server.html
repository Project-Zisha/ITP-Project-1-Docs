Detailed Description

A class for managing WebSocket connections.

This class manages WebSocket connections using a QWebSocketServer and a QList of QWebSocket objects. It provides methods for handling incoming messages, establishing new connections, and broadcasting messages to all connected clients.

Note
Code 200 - Connection established Code 201 - Received Message Code 202 - Received Binary Message

example_websocketserver_usage.cpp.
